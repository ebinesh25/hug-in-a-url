## How to run it

```bash
# 1. Get the code
git clone https://github.com/ebinesh25/hug-in-a-url.git
cd hug-in-a-url

# 2. Install stuff
npm install

# 3. Start in dev mode
npm run dev
```

To make a production build:

```bash
npm run build     # files go to /dist
npm run preview   # test the build
```
